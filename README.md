# Requirements
* Apache >= 2.0
* PHP >= 5.4
* MySQL >= 5
* Zend Framework = 1.12.9
* Navegador Web com suporte a JavaScript

# Recommended modules
* Apache Rewrite Rule
* PHP OpenSSL
* PHP MySQL
* PDO MySQL
* cURL

# Installation
* Renomeie o arquivo "application.ini.bkp", que está em "application/configs", para "application.ini" e informe os dados de conexão
